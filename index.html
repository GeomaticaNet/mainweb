Sobre mí

Hola! soy Matias Sendra, fundador de GeomaticaNet, sysadmin especializado en Sistemas de Información Geográfica con mas de 15 años de experiencia. Me apasiona el Web Mapping, el desarrollo y la gestión de proyectos SIG.

En GeomaticaNet diseñamos aplicaciones a medida del cliente ya sea particulares como empresas ofreciendo servicio de desarrollo web corporativo y diseño de aplicaciones geoespaciales utilizando las últimas tecnologías open source. Desde la instalación y configuración de la infraestructura en servidores hasta los detalles de la aplicación.

Nuestros servicios

Servicios

Gestión de proyectos SIG

Análisis y gestión de datos geoespaciales, conversión, digitalización y procesamiento de datos ráster y/o vectorial. Elaboración y diseño de bases de datos geoespaciales.

Desarrollo Web Mapping

Desarrollo de aplicaciones utilizando tecnología open source como Leaflet, Mapbox, OpenLayers, Geoserver, Postgis, Turf.js. Soluciones integrales para empresas e instituciones gubernamentales.

Análisis Geoespacial

Análisis de redes. Geomarketing para optimización de cobertura de servicios. Estudios de impacto ambiental. Estudios con modelos digitales de elevación (MDE) para planificación urbana y territorial, estudios de hidrología y minería.

Teledetección

Obtención y tratamiento de imágenes satelitales para análisis de toma de decisiones en agricultura, desastres naturales, análisis de suelos, atmósfera, etc.

Desarrollo Web

Diseño web responsivo y desarrollo de aplicaciones a medida tanto para escritorio como para dispositivos móviles, utilizando las ultimas tecnologías web.

Cartografía

Generación de cartografía impresa a diversas escalas. Planos, cartas, mapas turísticos, etc.

Diseño de escenarios 3D

Generación de escenarios 3D para uso en minería, simulación, arquitectura, etc, mediante motor de videojuegos Unreal Engine.
